DevEpos ADT Eclipse Plug-Ins

This update site just contains Plug-Ins built on ADT and no further Eclipse content. Use it to install into ABAP Development Tools (ADT).

Prerequsites
Installation
  1. In Eclipse, choose in the menu bar Help > Install New Software...
  2. Add the URL https://devepos.com/eclipse/2020-06
  3. Press Enter to display the available features.
  4. Select the features you want to install and choose Next.
  5. On the next wizard page, you get an overview of the features to be installed. Choose Next.
  6. Confirm the license agreements and choose Finish to start the installation.
